About HomePilot

HomePilot is a production-oriented, fully open-source GenAI application designed to run entirely on your own hardware. It unifies chat, image generation, editing, video creation, and AI-powered storytelling into a single continuous conversation — no cloud dependency, no data leakage, no vendor lock-in.

Built as the "Home Edition" of a broader enterprise vision, it is optimized for local machines with NVIDIA GPUs while remaining extensible enough to power professional content-creation workflows, intelligent AI agents, and team collaboration.

  • Privacy-first: all processing stays on your machine
  • Self-hosted: Docker Compose orchestration for easy deployment
  • Extensible: 160+ API endpoints, MCP tool servers, and A2A agents
  • Community-driven: Apache 2.0 licensed, contributions welcome
