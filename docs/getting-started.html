Frequently asked questions

Do I need to know how to code?
Not at all. HomePilot is a regular desktop app — download, install, and use it like any other program. No terminal or programming knowledge required.
Is it really free?
Yes. HomePilot is open-source under the Apache 2.0 license. There are no subscriptions, no premium tiers, and no telemetry. You may need an API key from OpenAI or Anthropic (which have their own pricing), or you can use Ollama to run free, local AI models.
What is Docker and why do I need it?
Docker is a tool that packages software so it runs the same way on every computer. HomePilot uses it behind the scenes to run its AI services. You don't need to learn Docker — the app will guide you through installing Docker Desktop if you don't already have it.
Can I use it without an internet connection?
Yes, if you use Ollama with locally downloaded models. Once the models are on your machine, everything runs offline. Cloud providers (OpenAI, Anthropic) require an internet connection.
Where is my data stored?
Everything stays on your computer. Conversations, images, videos, and settings are stored locally in a Docker volume. Nothing is sent to external servers.
How do I update HomePilot?
Each time you open HomePilot, it checks for updates in the background. If a new version is available, you'll get a notification — click it to update with one click. You can also check manually from the system tray menu. Your data and settings are always preserved.
