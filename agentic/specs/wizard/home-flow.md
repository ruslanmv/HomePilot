# Wizard flow: Home

## Goal
Make agent creation feel like a consumer assistant setup (no MCP jargon).

## Steps (1 screen)
1. **Name** (optional)
2. **Style**: Friendly / Neutral / Focused
3. **Done**

## Behind the scenes
Automatically binds:

* Tool bundle: `Personal Assistant Tools` (virtual server: `hp-home-default`)
* A2A agent: `Everyday Assistant`

No tool list is shown.
